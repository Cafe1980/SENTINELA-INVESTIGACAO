# 🛡️ SISTEMA SENTINELA - INTELIGÊNCIA & CUSTÓDIA
**Framework de Apoio à Investigação Forense e OSINT**

### 🚀 Funcionalidades:
*   **⛓️ Cadeia de Custódia:** Geração de Hash SHA-256 automático.
*   **🔎 Hub OSINT:** Dorks integradas para buscas governamentais.
*   **🔒 100% Local:** Segurança orgânica absoluta (Intranet).

### ⚙️ Instruções de Uso:
1. Baixe o ZIP do repositório.
2. Certifique-se de ter o **Node.js** instalado.
3. No terminal da pasta, execute: `node server.js`.
4. Acesse via navegador em `http://localhost:3000`.
